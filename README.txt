INSTRUCTIONS ON HOW TO OPERATE THE SYSTEM

1. DOWNLOAD AND INSTALL ORACLE SQL DEVELOPER and 11G ORACLE
	(This will be used to run the system)
2. OPEN ALL SQL IN THE FOLDER DELIVERABLES EXCEPT XE1.SQL AND PUT/IMPORT THEM AS PROCEDURES
	(These are all the procedures that will be run in the system)
3. SEPARATE THE FIRST SHEET IN FINAL.XLS AS CSV AND IMPORT IT AS TABLE IN ORACLE SQL DEVELOPER
	(This will be the input for the system)
4. OPEN AND RUN XE1.SQL
	(This will run all the procedures step by step as stated in the system architecture)

NOTE: ALL INPUTS AND OUTPUT ARE IN THE FINAL.XLS
